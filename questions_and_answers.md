###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: `The correct answer is B: Hi there, undefined.`



<i>Explanation: In the function sayHi, there is a parameter name that is not given a value when calling the function. When you call sayHi() without providing an argument, name becomes undefined inside the function, resulting in "Hi there, undefined" being returned and logged to the console.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: `The correct answer is C: 3.`



<i>Explanation:The code uses forEach to iterate through the nums array, incrementing count for each truthy value (non-zero). Since there are three truthy values (1, 2, and 3) in the array, count becomes 3, which is logged to the console.</i>

</p>
</details>
